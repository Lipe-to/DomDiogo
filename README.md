
# 🎓 Dom Diogo - Gestão Escolar Inteligente
Dom Diogo é uma plataforma educacional desenvolvida para centralizar a gestão acadêmica, 
unindo eficiência administrativa com uma experiência intuitiva para professores e alunos.


# Sobre o Projeto

 A plataforma Dom Diogo nasceu da necessidade de simplificar processos burocráticos no ambiente escolar.
 Com uma interface interativa, o sistema oferece funcionalidade específicos para cada nível de acesso, 
 garantindo que os dados pedagógicos se transformem em insights para a tomada de decisão.

# Níveis de Acesso e Funcionalidades

O sistema é dividido em três pilares fundamentais, cada um com ferramentas personalizadas:
- Administrador - Controle de Usuários:
    - Cadastro, edição e remoção de alunos e professores.
    - Visão Macro: Funcionalidades geral. Consegue adicionar, remover editar as notas, observações. 
    - Monitoramento: Acompanhamento de métricas de desempenho da instituição.
- Professor - Gestão de Notas:
    - Lançamento ágil de avaliações e observações individuais.
    - Análise de Turma.
    - Espaço para observações qualitativas sobre o comportamento e evolução do aluno.
- Aluno - Painel de Notas:
    - Visualização em tempo real do desempenho acadêmico.
    - Status de Aprovação: Cálculo automático de situação (Aprovado/Reprovado).
    - Boletim Digital: Download do boletim oficial em formato PDF com um clique.
